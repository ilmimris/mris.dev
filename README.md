## Running Locally


```sh-session
git clone https://github.com/ilmimris/mris.dev.git
cd mris.dev
```


Create a `.env` file similar to [`.env.example`](https://github.com/ilmimris/mris.dev/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```
